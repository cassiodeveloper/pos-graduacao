Descrição do curso

Este curso vai te dar a capacidade de tornar os seus softwares mais seguros, não importa em qual fase do desenvolvimento você atue, seja como analista de sistemas, desenvolvedor, tester, DBA, analista de infra, devops etc. Introduzir segurança da forma correta tem se tornado cada vez mais fundamental no mercado tecnológico atual, e conhecer cada etapa do ciclo de desenvolvimento seguro é fator crítico de sucesso para os seus projetos. Se você quer se tornar um Security Champion, ou mesmo um AppSec Engineer este curso é para você.

Este curso de extensão oferece uma abordagem prática e objetiva para implementar segurança no desenvolvimento de software. Os participantes aprenderão a identificar e mitigar vulnerabilidades comuns, integrar segurança no ciclo de desenvolvimento (SDLC) e aplicar boas práticas de codificação segura, garantindo aplicações mais protegidas contra ataques.

O que você vai aprender?

  • Modelagem de Ameaças
  • Cenários reais
  • Segurança em tempo de desenvolvimento SDLC
  • Segurança em CI/CD
  • Segurança em ambientes
  • Monitoramento de aplicações
  • Gestão de vulnerabilidades
  • Programa de Desenvolvimento Seguro

Desenvolva com segurança: código forte, aplicações resilientes.
